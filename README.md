# Suumo分析预测[![PyPI Version]][PyPI] [![Build Status]][Travis CI] [![Coverage Status]][Coverage]
## 简介
本程序是一个简单的网页爬虫。爬取Suumo.jp网站的物件信息，并使用RandomForest模型进行两年总费用的预测。
## 文件结构
- Scraping.py			-爬虫
	 Data_Cleaning.py		-数据清洗
	 Data_OneHotEncode.py	-特征数据独热编码
	 Train_RF.py			-训练模型
	 Data_Query.py			-输出数据查询
	 Data_Graph.py			-图形化
